  • 00
    Atonement
    (Indie-Game, 2019)

    The Atonement is a game created using game engine Unity with internal C# programming language. The most difficult parts of development are pathfinding system (AI) and game/level-design.

    The plot of the game revolves around the main character - a relatively ordinary, average person who suffers from constantly haunting nightmares in which he sees himself as the protagonist of various fairy tales that always end badly for the character.

    The dreams in which he participates characterize the sins or actions he has committed throughout his life, and their list is not so small - greed, pride, envy, anger, despair.

    This list can be continued for a long time, but whether the hero will be able to get rid of the nightmares that torment him and cleanse himself of his deeds or not - you will have to find out during the game.

  • 01
    Electronic Diary
    (School, 2020)

    The app is an school journal for schools that aims to solve the problem of quickly transferring information from physical to electronic media.

    The app uses machine learning methods (e.g. OCR to transfer data) and has two main user roles: teacher and student.

    Teachers can set grades, view the class list, add comments to students` tasks, use special chat, make lesson plans, view schedules, and have quick access to grades.

    Students can view their grades, use chat to talk with teachers, receive statistics and view schedules.

    The app is developed for PC using Python and its` different libraries.

  • 02
    KazanTourism
    (WiP, 2023)

    The objective of this application is to serve as a user-friendly and invaluable tool for tourists and individuals eager to explore new destinations. The primary goal is to seamlessly match users with places of interest, utilizing applied filters and search queries.

    Additionally, the application aims to equip users with essential information that enables informed decision-making and efficient trip planning. A significant aspect of the application centers around empowering users to contribute by adding new locations and moderating already presented, thus expanding the breadth and diversity of available options and clarifying the description of places.

    This feature ensures that the application remains a comprehensive resource, allowing users to continually discover fresh and exciting places to visit. After all, the app will remain flexible and reliable, since the users will update the data on the run.
